Recentemente aggiornata per usare le più recenti tecnologie android,
questa sublime applicazione open source vanta le seguenti funzioni:

* Supporto per Dash Clock (si, funziona ancora)
* Notifiche, anche con ripetizione settimanale
* Selezione della lingua preferita
* Widget ridimensionabile e configurabile
* Sincronizzazione automatica con Google Tasks
* Protezione delle note con password
* Liste personalizzate tra le quali si possono spostare le note
* Liste dove si possono riordinare a piacere le note
* Ricerca tra le note
* Disponibili temi chiari e scuri
* Importazione & esportazione su file JSON
* Sincronizzazione con file nella memoria locale

Se vuoi migliorare quest'app, o condividere qualche idea, visita la pagina su github.


Una possibile alternativa è https://f-droid.org/packages/com.orgzly
Orgzly è un'app per le note in sviluppo attivo che può importare le vecchie note di quest'app
